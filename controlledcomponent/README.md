# 🎛️ React Controlled Component Example

This project demonstrates how to create a **controlled component** in React using a textarea input.

---

## 🚀 Features

* Functional React component
* Using `useState()` hook
* Controlled textarea input
* Real-time data display
* Handling user input dynamically

---

## 🧠 Concept: Controlled Component

A controlled component is a form element whose value is controlled by **React state**.

* React manages the input value
* UI always reflects the state
* Single source of truth = React state

---

## 🔥 How it works

* User types in the textarea
* `onChange` event captures input
* State is updated using `useState()`
* Updated value is displayed instantly

---

## 📌 Behavior in This Project

* Text entered in textarea is stored in state
* The same text is displayed below in real-time
* Input and UI stay in sync

---

## 🎯 Key Understanding

* `useState()` is used to store input value
* `onChange` updates the state
* Controlled components give full control over form data

---

## ⚠️ Important Notes

* Always use `value` with controlled inputs (best practice)
* Helps in validation and form handling
* Avoid mixing controlled and uncontrolled inputs

---

## 📌 Common Use Cases

* Forms and user input
* Validation and error handling
* Dynamic UI updates
* Real-time previews

---

# 🔍 React useSearchParams Example

This project demonstrates how to use **query parameters** in React using the `useSearchParams` hook from **React Router DOM**.

---

## 🚀 Features

* Using `useSearchParams()` hook
* Setting query parameters dynamically
* Reading values from URL
* Updating UI based on URL parameters
* Simple state management with hooks

---

## 🧠 Concept: Query Parameters

Query parameters are values added to the URL after `?`.

Example:

```
http://localhost:3000/?name=Rutwik&age=50
```

They are commonly used to:

* Pass data in URLs
* Filter/search results
* Maintain application state

---

## 🔥 useSearchParams Hook

`useSearchParams()` allows you to:

* Read query parameters
* Update query parameters

---

## 📌 Behavior in This Project

* Clicking the button sets query parameters in the URL
* URL updates with `name` and `age`
* Values are read from the URL
* Data is displayed on the screen

---

## ⚠️ Important Notes

* Query parameters are always strings
* URL updates without page reload
* Useful for sharing application state via URL

---

## 📌 Real Flow

* User clicks button
* Query parameters are added to URL
* React reads updated parameters
* UI updates with new values

---

* Learn to read and update URL data
